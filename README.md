## Purpose of creation

1. 肝臓を大事にするべくアルコール量に気を付けたい。
2. 飲んだアルコール飲料のアルコール量を計算したい。
3. 制限アルコール量まで、あとどれくらい飲めるか知りたい。

## Other

1. 純アルコール量計算式は厚生省のHPを参考に下記で計算しました。  
   摂取量(ml) × アルコール濃度（度数/100）× 0.8（アルコールの比重）
